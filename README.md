Bootstrap Gallery - bootstrap-gallery
=============

Simples plugin para exibição de imagens em uma caixa modal. Baseado em jQuery e Bootstrap Modal.

Requirements
------------

* [Bootstrap](http://getbootstrap.com)
* [jQuery](http://www.jquery.com)
